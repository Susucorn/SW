문제 정의
    정수 2개와 연산자 4개 (+, -, *, /) 중 1개를 입력 받아서 사칙연산을 수행하는 프로그램 작성

문제 분석
    1. 변수
     정수1(num1), 정수2(num2), 연산자(op)
    2. num1, num2는 정수값으로 받고 op는 문자값으로 받기

수식
num1 op num2

알고리즘
    1. 변수 선언
    2.키보드로 정수값 받기, 연산자는 문자값으로 받기
    3. 선택한 연산자에 맞게 결과 수행
        if문을 사용하여 선택한 연산자에 맞는 연산하여 결과 출력
        마지막으로 받은 연산자는 else 결과 출력
            if op=='+':
                num1+num2
            elif op=='-':
                num1-num2
            elif op=='*':
                num1*num2
            else:
                num1/num2
